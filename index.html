💳 POC Integração de Pagamentos

Guia completo de integração para processar pagamentos via Cartão de Crédito e Pix QR Code com Getnet, Mercado Pago e Pagar.me

🔴 Getnet 🔵 Mercado Pago 🟢 Pagar.me ⚡ Pix 💳 Cartão 🔔 Webhooks

Bem-vindo ao Guia

Esta documentação é um Proof of Concept (POC) completo para integração de pagamentos nos nossos sistemas. Cobrimos três integradores (gateways/adquirentes) disponíveis no Brasil, com análise comparativa de taxas, facilidade de integração e recomendações técnicas.

🔴

Getnet

API do Santander — OAuth 2.0, cartão, Pix, antifraude incluso

🔵

Mercado Pago

Checkout Transparente via Orders API — D+0, Pix 0%, SDK maduro

🟢

Pagar.me

API mais robusta do Brasil — PSP tech, split nativo, checkout hosted

Como navecar

Use a sidebar à esquerda para navegar entre as páginas:

  1. Comparativo — entenda qual integrador escolher
  2. Stack & Infra — como estruturar o backend
  3. Banco de Dados — schema e modelo de dados
  4. Getnet — integração completa com a Global API
  5. Mercado Pago — integração com a Orders API
  6. Pagar.me — integração com a API v5
📊 Recomendação rápida Para início de operações com foco em boas taxas e fácil API: Mercado Pago (D+0, Pix 0%, onboarding rápido) ou Pagar.me (API mais robusta, split nativo). Getnet é sólido se já há relação com o Santander.
